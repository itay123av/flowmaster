# ⚡ FlowMaster

מערכת ניהול זמן ופרודקטיביות מקיפה — אפליקציית HTML יחידה, כל הנתונים נשמרים מקומית בדפדפן.

## ✨ פיצ'רים

### 🏠 דשבורד
- סטטיסטיקות חיות וציון פרודקטיביות יומי (0–100)
- מבט שבועי עם מעקב תפוקה לפי יום
- טיפ יומי וגישה מהירה לרוטינות בוקר/ערב

### ✅ ניהול משימות — מטריצת אייזנהאואר
- 4 רביעים: דחוף+חשוב, חשוב, דחוף, וכל השאר
- Drag & Drop בין רביעים
- תיוגים והשלמה במחיקה אחת

### 📅 Time Blocking
- לוח זמנים יומי 06:00–23:00
- בלוקים צבעוניים עם שעות התחלה/סיום
- לחיצה על שעה ליצירת בלוק מהיר

### 🍅 פומודורו
- 4 מצבים: 25 דק' עבודה, 5 דק' הפסקה, 15 דק' הפסקה ארוכה, 50 דק' עבודה עמוקה
- טיימר עגול עם גרדיאנט וצלצול סיום
- **מצב פוקוס מלא־מסך**
- **רץ גם כשהטאב ברקע** (Web Worker + timestamp-based timing)
- כותרת הטאב מציגה את הזמן הנותר

### 🎯 יעדים
- 6 טווחים: יומי / שבועי / חודשי / רבעוני / שנתי / חיים
- אבני דרך עם פרוגרס בר אוטומטי
- תאריך יעד ותיאור ה-Why

### 🔥 הרגלים
- תצוגת רשת 30 ימים בסגנון GitHub contributions
- מעקב streaks אוטומטי
- עיקרון "אל תשבור את השרשרת"

### 📔 יומן רפלקציה
- 10 שאלות רפלקציה אקראיות
- מעקב מצב רוח (1–5)
- תבניות מהירות: הצלחות, למידה, תודה, פוקוס

### 📊 אנליטיקה
- גרף עמודות 14 ימים
- תובנות חכמות: היום/שעה הכי פרודקטיביים שלך
- חלוקה לפי רביעי אייזנהאואר

## ⌨️ קיצורי מקשים

| מקש | פעולה |
|---|---|
| `1`–`8` | ניווט בין מודולים |
| `Q` או `/` | משימה מהירה |
| `Ctrl+K` | חיפוש |
| `Space` | הפעלה/השהיה של פומודורו |
| `Esc` | סגירת חלונות |

## 🚀 שימוש

פשוט פתח את `index.html` בדפדפן. הכל מקומי — אין שרת, אין חשבון, אין תלויות.

הנתונים נשמרים אוטומטית ב־`localStorage`. אפשר לייצא ולייבא גיבוי JSON מהסרגל העליון (⬇⬆).

## 🛠 טכנולוגיות

- HTML/CSS/JS וניל — קובץ אחד, אפס תלויות
- `localStorage` להתמדה
- Web Worker לטיימר שלא נעצר ברקע
- Web Audio API לצלצול סיום
- RTL מלא, פונט Heebo
